小红书文案合规检测与优化工具。

**输入**: 用户提供的文案内容或描述

**检测维度**

1. **合规检测**
   - 敏感词检测
   - 广告法合规
   - 平台规则符合

2. **调性优化**
   - 小红书风格匹配
   - 情绪价值提升
   - 种草话术优化

3. **传播优化**
   - 标题吸引力
   - 话题标签建议
   - 排版格式优化

4. **卖点提炼**
   - 核心卖点突出
   - 差异化优势展示
   - 情感连接建立

**Steps**

1. **文案分析**
   - 读取文案内容
   - 分析文案结构
   - 识别核心卖点

2. **合规检测**
   - 检查敏感词
   - 验证广告合规
   - 确认平台规则

3. **优化建议**
   - 提供修改方案
   - 优化文案结构
   - 添加话题标签

**输出**

- 合规检测报告
- 优化后的文案
- 传播建议

**使用示例**

```
@xiaohongshu-content-checker 检测参赛帖文案
@xiaohongshu-content-checker 优化星宠漫游馆宣传文案
@xiaohongshu-content-checker 生成小红书种草文案
```
