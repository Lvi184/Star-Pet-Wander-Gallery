前端交互动效生成工具。

**输入**: 用户指定要添加动效的页面、组件或具体动效类型

**动效类型**

1. **骨架屏动画**
   - 页面加载占位动画
   - 数据加载状态展示
   - 渐入渐出效果

2. **视差滚动**
   - 多层背景视差漂移
   - 元素滚动视差
   - 星空背景动效

3. **按钮微动画**
   - 悬停缩放效果
   - 点击反馈动画
   - 波纹扩散效果

4. **页面过渡**
   - 页面切换动画
   - 路由过渡效果
   - 模态框弹出动画

5. **组件动效**
   - 宠物状态渐变
   - 日记翻页动画
   - 卡片hover效果
   - 进度条动画

**Steps**

1. **分析目标组件**
   - 读取目标文件内容
   - 识别现有样式结构
   - 确定动效插入位置

2. **生成动效代码**
   - 根据需求选择动效类型
   - 生成 CSS/JavaScript 代码
   - 确保与现有样式兼容

3. **插入动效代码**
   - 修改目标文件
   - 添加必要的 CSS 类
   - 编写动画逻辑

**输出**

- 修改后的文件内容
- 动效预览说明
- 样式代码片段

**使用示例**

```
@interaction-design 给宠物状态面板添加渐变动效
@interaction-design 给星空背景添加视差漂移
@interaction-design 给日记页面添加翻页动画
```
